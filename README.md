# WH Calculator

A single-file work-hours tracker that runs entirely in the browser — no server, no dependencies, no install.

## Features

- **Daily tracking** — log in/out times, live elapsed timer, early-leave targets
- **Weekly dashboard** — progress rings, surplus/deficit per day, override markers (Half Day, Holiday, All Day Leave)
- **Settings** — configurable shift times, weekly target (H + M), multi-select early-leave days
- **Import / Export** — structured JSON backup (`wh_backup_v2`) for full data portability
- **Dark mode** — auto-detects system preference, toggle in the nav

## Usage

Just open `index.html` in any modern browser. All data is stored in `localStorage`.
